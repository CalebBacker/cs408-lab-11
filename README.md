# Web Dev Starter Code

## Overview

This is a simple web application that utilizes AWS to for the backend services. 
The frontend is built using HTML, CSS, and JavaScript. 
The application itself is an inventory management system that allows users to add, view,
and delete items from their inventory.

To use, run npm install in the terminal, than use either live preview in vs code, or run the html file in intellij.
Then use the buttons to input, delete, and load data.
